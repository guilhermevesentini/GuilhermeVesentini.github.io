Instalttec

Since 2019 Instalttec is being develop for accountant profesisonals

Me and my collegue Hugo Machado had been working on this project to facilitate accountance profesisonals lifes, looking foward for, organization and a faster working enviroment.

This tool we created mainly offer the Conversor service for the moment, this tool is able to convert an XML document into a excel document. That idea came from the dificulty it has to check item by item and informations inside a xml document which is not very clear, to analyze takes long hours working when it has to be done in several documentations. Once you have Instalttec services we will provide all these documentations organized and easy to access in excel format, independent the number of files is necessary to verification.

In this project for the front end Development:
Html5, CSS3, React JS, Javascript, Jquery, Illustrator, Bootstrap.

Hugo Machado is responsible for the Back end Development.
